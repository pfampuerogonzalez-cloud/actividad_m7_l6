# ACTIVIDAD MODULO 7, LECCIÓN 6

### ¿COMO FUNCIONA EL FLUJO COMPLETO DE UNA OPERACIÓN CRUD?

- R. EL FLUJO FUNCIONA CONECTANDO LA INTERFAZ DE USUARIO, EL BACKEND O API Y LA BASE DE DATOS.

### ¿QUÉ APRENDISTE SOBRE EL ENRUTAMIENTO Y LOS PARÁMETROS DINÁMICOS EN URLS?

- R. QUE EL ENRUTAMIENTO Y PARÁMETROS DINÁMICOS SON EL SISTEMA DE NAVEGACIÓN DE LA APLICACIÓN WEB Y PERMITEN QUE UNA SOLA PLANTILLA DE DISEÑO MUESTRE DETERMINADO CONTENIDO SEGÚN LA URL QUE GATILLE EL USUARIO.
